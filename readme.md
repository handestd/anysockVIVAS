## Run Locally

Clone the project

```bash
  git clone https://github.com/handestd/anysockVIVAS.git
```

Run docker compose

```bash
  docker-compose.yaml
```

Import Postman manually

```bash
  attachment/AnysockVIVAS API.postman_collection.json
```
